# 代码质量命令

这个命令执行全面的代码质量检查，确保代码符合最高的质量标准。

## 质量检查项目

### 1. 类型安全
- Java 编译检查
- 类型定义完整性
- 接口契约验证

### 2. 代码规范
- 基本命名规范验证
- 编码标准合规

### 3. 测试质量
- 单元测试完整性
- 边界条件测试

## 执行命令

```bash
# 运行所有质量检查
/code-quality

# 运行特定类型检查
/code-quality --type-check
```

通过这个命令，您可以确保代码符合所有质量标准和最佳实践要求。
