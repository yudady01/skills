# 知识库总览（knowledge）

本目录下是整个知识库的根目录，按不同业务/主题划分为多个领域子目录。

- Safety Knowledge/
  - 用途：安全相关知识与文档，涵盖常见 Web 安全漏洞及防护方案。
- Financial Report Data/
  - 用途：金融与上市公司财报类数据，主要是按公司和季度划分的 PDF 报告。
- E-commerce Data/
  - 用途：电商业务相关的数据文件，如员工、客户、库存和订单等结构化数据。
 - AI Knowledge/
   - 用途：人工智能行业与技术相关报告与研究资料，用于检索 AI 技术发展、应用场景及治理趋势。
