# 網域名稱腦力激盪器

這個 skill 透過生成創意選項並檢查哪些網域實際可用來註冊，協助您為專案找到完美的網域名稱。

## 何時使用此 Skill

- 開始新專案或公司
- 推出產品或服務
- 建立個人品牌或作品集網站
- 為現有專案重新命名
- 為副專案註冊網域
- 當您的首選被佔用時尋找可用的替代方案

## 此 Skill 的功能

1. **了解您的專案**：分析您正在建構的內容和目標對象
2. **生成創意名稱**：建立相關、令人難忘的網域選項
3. **檢查可用性**：驗證哪些網域在多個頂級網域中實際可用
4. **多種擴展名**：建議 .com、.io、.dev、.ai、.app 等
5. **提供替代方案**：如果首選被佔用，提供變體
6. **品牌見解**：解釋為何某些名稱效果好

## 網域命名技巧

### 什麼是好的網域

✓ **簡短**：理想情況下少於 15 個字元
✓ **令人難忘**：易於回憶和拼寫
✓ **可發音**：可以在對話中說出
✓ **描述性**：暗示您的功能
✓ **可品牌化**：足夠獨特以脫穎而出
✓ **無連字號**：口頭分享更容易

### 頂級網域指南

- **.com**：通用、受信任、適合企業
- **.io**：科技新創公司、開發者工具
- **.dev**：以開發者為中心的產品
- **.ai**：AI/ML 產品
- **.app**：行動或網頁應用程式
- **.co**：.com 的替代方案
- **.xyz**：現代、創意專案
- **.design**：創意/設計公司
- **.tech**：科技公司

## 範例工作流程

### 新創公司推出
1. 描述您的新創公司想法
2. 獲得跨頂級網域的 10-15 個網域建議
3. 審查可用性和定價
4. 選擇前 3 個最愛
5. 立即註冊

### 個人品牌
1. 分享您的姓名和職業
2. 獲得變體（firstname.com、firstnamelastname.dev 等）
3. 也檢查社群媒體帳號的可用性
4. 在各平台上註冊一致的品牌

### 產品命名
1. 描述產品和目標市場
2. 獲得創意、可品牌化的名稱
3. 檢查商標衝突
4. 驗證網域和社群媒體可用性
5. 與目標受眾測試名稱

## 成功技巧

1. **快速行動**：好的網域很快就會被佔用
2. **註冊變體**：獲取 .com 和 .io 以保護品牌
3. **避免數字**：口頭溝通困難
4. **檢查社群媒體**：確保 @username 也可用
5. **大聲說出來**：測試是否容易發音
6. **檢查商標**：確保沒有法律衝突
7. **長遠思考**：5 年後它仍然有意義嗎？

## 定價背景

在建議網域時，我會注意：
- 標準網域：每年約 $10-15
- 高級頂級網域（.io、.ai）：每年約 $30-50
- 已佔用網域：如果有列出市場價格
- 高價網域：數百到數千美元
