# Challenge Statistics

View detailed statistics for your active challenge including progress, streaks, patterns, and achievements.

## Usage

```
/streak-stats
```

## Output Sections

### Progress
- Total sessions
- Days since start
- Completion rate
- Time invested (if tracked)

### Streaks
- Current streak
- Longest streak
- Average gap between check-ins

### Patterns
- Most active day of week
- Best time of day
- Average session length

### Achievements Earned
- Streak badges (First Flame, On Fire, Unstoppable, Diamond)
- Milestone badges (First Step, Dedicated, Centurion)
- Special badges (Connected, Comeback)

### Backlog Status
- Completed items
- In progress items
- Pending items

## Instructions

Follow Flow 5: Progress Statistics from the `ccc-skills:streak` skill.

1. Read active challenge's `challenge-config.md`
2. Calculate all statistics
3. Read `backlog.md` for item counts
4. Format and display comprehensive stats
