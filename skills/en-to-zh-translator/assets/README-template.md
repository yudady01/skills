# 翻译模板

这是一个翻译模板文件，用于演示翻译技能的输出格式。

## 模板结构

### 代码块保持不变
```python
def example_function():
    # 这个注释会被翻译
    return "Hello, World!"  # 但代码本身保持不变
```

### 技术术语保留
- 使用 `API` 而不是 "应用程序接口"
- 使用 `JSON` 而不是 "JavaScript 对象表示法"
- 函数名 `calculate_fibonacci` 保持不变

### 数学表达式保留
- 时间复杂度: $O(n)$
- 空间复杂度: $O(1)$
- 第 $n$ 个元素

### 格式保持
- **粗体文本** → **粗体文本**
- *斜体文本* → *斜体文本*
- `行内代码` → `行内代码`

## 链接处理
- [技术文档](https://example.com) → [技术文档](https://example.com)
- 外部链接保持不变
