# 英中技术翻译器

专业的英中翻译工具，专门处理技术和编程内容。

## 功能特性

- 翻译代码提示、技术文档、AI提示和开发材料
- 保持格式、代码块、技术术语和LaTeX数学表达式不变
- 为软件开发环境保持专业语调和技术准确性

## 使用方法

在 Claude Code 中使用 `en-to-zh-translator:skills` 技能来翻译技术内容。

## 支持的内容类型

- AI 提示和指令
- 技术文档
- 编程教程和指南
- 代码注释和技术说明
- 以开发者为重点的内容

## 翻译原则

- **代码块**：保持所有代码不变
- **技术术语**：保留原始英文术语
- **变量名**：不翻译函数名、变量名或API
- **文件扩展名**：保持不变
- **数学表达式**：保持所有LaTeX数学表达式不变
