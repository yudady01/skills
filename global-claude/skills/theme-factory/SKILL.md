# 主题工厂技能

此技能提供精心策划的专业字体和颜色主题集合，每个主题都有精心挑选的调色板和字体配对。一旦选择了主题，就可以应用于任何工件。

## 目的

要为演示幻灯片应用一致、专业的样式，请使用此技能。每个主题包括：
- 带有十六进制代码的连贯调色板
- 标题和正文的互补字体配对
- 适合不同场景和受众的独特视觉标识

## 使用说明

要为幻灯片或其他工件应用样式：

1. **显示主题展示**：显示 `theme-showcase.pdf` 文件，让用户直观地查看所有可用主题。不要对其进行任何修改；仅显示文件供查看。
2. **询问选择**：询问要将哪个主题应用到幻灯片
3. **等待选择**：获得对所选主题的明确确认
4. **应用主题**：选择主题后，将所选主题的颜色和字体应用到幻灯片/工件

## 可用主题

以下 10 种主题可用，每个都在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths（海洋深处）** - 专业而宁静的海洋主题
2. **Sunset Boulevard（日落大道）** - 温暖而充满活力的日落色彩
3. **Forest Canopy（森林树冠）** - 自然而沉稳的大地色调
4. **Modern Minimalist（现代极简）** - 干净而现代的灰度色彩
5. **Golden Hour（金色时刻）** - 丰富而温暖的秋季调色板
6. **Arctic Frost（北极霜冻）** - 清凉爽快的冬季灵感主题
7. **Desert Rose（沙漠玫瑰）** - 柔和而精致的尘土色调
8. **Tech Innovation（科技创新）** - 大胆而现代的科技美学
9. **Botanical Garden（植物园）** - 清新有机的花园色彩
10. **Midnight Galaxy（午夜星系）** - 戏剧性而深邃的宇宙色调

## 主题详情

每个主题都在 `themes/` 目录中定义，包含完整的规格，包括：
- 带有十六进制代码的连贯调色板
- 标题和正文的互补字体配对
- 适合不同场景和受众的独特视觉标识

## 应用流程

选择首选主题后：
1. 从 `themes/` 目录读取相应的主题文件
2. 在整个幻灯片中一致地应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉标识

## 创建您自己的主题
要处理现有主题都不适用于工件的情况，请创建自定义主题。根据提供的输入，生成一个与上述主题类似的新主题。给主题一个类似的名称，描述字体/颜色组合代表的内容。使用任何提供的基本描述来选择适当的颜色/字体。生成主题后，显示它以供审查和验证。之后，按照上述说明应用主题。
