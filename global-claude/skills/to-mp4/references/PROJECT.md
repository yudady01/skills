# to-mp4 项目概述

MOV 转 MP4 视频转换器 - 智能压缩视频并保持良好画质。

## 简介

to-mp4 是一个基于 Python + FFmpeg 的视频转换工具，专门用于将 MOV 格式视频转换为 MP4 格式。它提供了智能压缩选项，可以显著减小文件体积同时保持可接受的画质。

## 核心功能

- **格式转换**: MOV → MP4
- **智能压缩**: H.264/H.265 编码支持
- **交互模式**: 3 种预设压缩方案
- **画质控制**: CRF 18-28 可调
- **分辨率调整**: 自动保持宽高比缩放
- **音频处理**: AAC/MP3 编码支持
- **Web 优化**: Faststart 流式播放

## 技术栈

| 组件 | 技术 |
|------|------|
| 编程语言 | Python 3.6+ |
| 视频处理 | FFmpeg 4.0+ |
| 视频编码 | H.264 (libx264), H.265 (libx265) |
| 音频编码 | AAC, MP3 |

## 设计理念

### 1. 简单优先
- 默认设置适合大多数场景
- 交互模式降低学习曲线
- 清晰的命令行选项

### 2. 灵活性
- 支持自定义编码参数
- 多种音频编码选项
- 分辨率自由调整

### 3. 性能与质量平衡
- CRF 模式保证画质
- 可选 H.265 高压缩率
- 合理的默认预设

## 使用场景

| 场景 | 推荐方案 |
|------|---------|
| 日常分享 | H.264 + CRF 23 |
| Web 上传 | H.264 + 降分辨率 |
| 存储优化 | H.265 + CRF 28 |
| 快速预览 | H.264 + fast preset |

## 压缩效果

基于实际测试数据：

| 输入 | 方案 | 输出 | 压缩率 |
|------|------|------|--------|
| 2GB MOV (4K) | H.264 CRF23 | ~300MB | 85% |
| 2GB MOV (4K) | H.264 + 1080p | ~150MB | 92.5% |
| 2GB MOV (4K) | H.265 CRF28 | ~200MB | 90% |
