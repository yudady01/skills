## Features

### Tablestakes (Everyone Has)
- Issue tracking with custom fields
- Kanban and list views
- Labels, priorities, assignees
- GitHub/GitLab integration
- Slack integration
- API access

### Differentiators (Their USPs)
- **Speed** - Keyboard-first, feels instant. "Fastest issue tracker" positioning.
- **Design quality** - Beautiful dark-first UI. Developer aesthetic.
- **Cycles** - Time-boxed sprints with auto-scheduling. Alternative to traditional sprints.
- **Roadmaps** - Visual roadmap builder, integrates with issues
- **Triage** - Dedicated inbox for new issues, prevents backlog chaos
- **Auto-archiving** - Old completed issues auto-archive

### Emerging (New Bets)
- **Linear Asks** (2024) - Customer feedback collection → issues
- **AI features** - Auto-categorization, duplicate detection
- **Initiatives** - Higher-level grouping above projects
