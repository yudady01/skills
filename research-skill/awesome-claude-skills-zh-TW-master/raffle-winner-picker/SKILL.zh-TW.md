# 抽獎獲獎者選擇器

這個 skill 從清單、試算表或 Google Sheets 中隨機選擇贈品和競賽的獲獎者。

## 何時使用此 Skill

- 舉辦社群媒體贈品活動
- 在活動中挑選抽獎獲獎者
- 隨機選擇調查或測試參與者
- 從競賽提交中選擇獲獎者
- 公平分配有限名額或資源
- 隨機團隊分配

## 此 Skill 的功能

1. **隨機選擇**：使用加密安全的隨機選擇
2. **多種來源**：適用於 CSV、Excel、Google Sheets 或純清單
3. **多位獲獎者**：可以挑選一位或多位獲獎者
4. **防止重複**：確保同一人不會贏兩次
5. **透明結果**：清楚顯示選擇過程
6. **獲獎者詳細資訊**：顯示獲獎者的所有相關資訊

## 使用方法

### 從 Google Sheets

```
從這個 Google Sheet 中挑選一個隨機列來選擇贈品獲獎者：
[Sheet URL]
```

### 從本地檔案

```
從 entries.csv 中挑選 3 位隨機獲獎者
```

### 從清單

```
從這個清單中挑選一位隨機獲獎者：
- Alice (alice@email.com)
- Bob (bob@email.com)
- Carol (carol@email.com)
...
```

### 多位獲獎者

```
從 contest-entries.xlsx 中挑選 5 位隨機獲獎者，
確保沒有重複
```

## 功能

### 公平選擇
- 使用安全的隨機數生成
- 無偏見或模式
- 透明的過程
- 可使用種子重複（用於驗證）

### 排除項目
```
挑選一位隨機獲獎者，排除之前的獲獎者：
Alice、Bob、Carol
```

### 加權選擇
```
根據「參賽次數」欄位以加權機率挑選獲獎者
（1 次參賽 = 1 張票）
```

### 候補者
```
從清單中挑選 1 位獲獎者和 3 位候補者
```

## 範例工作流程

### 社群媒體贈品活動
1. 將 Google Form 的參賽作品匯出到 Sheets
2. 「從 [Sheet URL] 中挑選一位隨機獲獎者」
3. 驗證獲獎者詳細資訊
4. 公開宣布並附時間戳

### 活動抽獎
1. 建立參加者姓名和電子郵件的 CSV
2. 「從 attendees.csv 中挑選 10 位隨機獲獎者」
3. 匯出獲獎者清單
4. 直接發送電子郵件給獲獎者

### 團隊分配
1. 擁有參與者清單
2. 「將這個清單隨機分成 4 個相等的團隊」
3. 審查分配
4. 分享團隊名單

## 技巧

- **記錄過程**：儲存時間戳和方法
- **公開宣布**：分享選擇細節以保持透明
- **檢查資格**：驗證獲獎者符合競賽規則
- **準備備案**：挑選候補者以防獲獎者不符合資格
- **匯出結果**：儲存獲獎者清單以供記錄

## 隱私與公平性

✓ 使用加密安全的隨機性
✓ 不可能操縱
✓ 記錄時間戳以供驗證
✓ 可提供種子供第三方驗證
✓ 尊重資料隱私
