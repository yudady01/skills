# 影片下載器

這個 skill 直接將 YouTube 和其他平台的影片下載到您的電腦。

## 何時使用此 Skill

- 下載 YouTube 影片以供離線觀看
- 儲存教育內容以供參考
- 存檔重要影片
- 獲取影片檔案以供編輯或重新利用
- 從平台下載您自己的內容
- 儲存會議演講或網路研討會

## 此 Skill 的功能

1. **下載影片**：從 YouTube 和其他平台獲取影片
2. **品質選擇**：讓您選擇解析度（480p、720p、1080p、4K）
3. **格式選項**：以各種格式下載（MP4、WebM、僅音訊）
4. **批次下載**：可以下載多個影片或播放清單
5. **中繼資料保留**：儲存標題、描述和縮圖

## 使用方法

### 基本下載

```
下載這個 YouTube 影片：https://youtube.com/watch?v=...
```

```
以 1080p 品質下載這個影片
```

### 僅音訊

```
將這個 YouTube 影片的音訊下載為 MP3
```

### 播放清單下載

```
下載這個 YouTube 播放清單中的所有影片：[URL]
```

### 批次下載

```
下載這 5 個 YouTube 影片：
1. [URL]
2. [URL]
...
```

## 重要注意事項

⚠️ **版權與合理使用**
- 僅下載您有權限下載的影片
- 尊重版權法和平台服務條款
- 用於個人、教育或合理使用目的
- 不要重新分發受版權保護的內容

## 技巧

- 如果需要較小的檔案大小，請指定品質（720p vs 1080p）
- 對於播客或音樂，使用僅音訊以節省空間
- 下載到專用資料夾以保持組織性
- 在慢速連接上下載前檢查檔案大小

## 常見使用案例

- **教育**：儲存教學和課程以供離線學習
- **研究**：存檔影片以供參考
- **內容創作**：從平台下載您自己的內容
- **備份**：在影片被刪除前儲存重要影片
- **離線觀看**：在沒有網路的情況下觀看影片
