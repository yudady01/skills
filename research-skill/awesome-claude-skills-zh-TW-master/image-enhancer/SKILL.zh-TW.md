# 圖片增強器

這個 skill 讓您的圖片和螢幕截圖看起來更好——更銳利、更清晰、更專業。

## 何時使用此 Skill

- 為部落格文章或文件改善螢幕截圖品質
- 在社群媒體上分享前增強圖片
- 為簡報或報告準備圖片
- 提升低解析度圖片
- 銳化模糊照片
- 清理壓縮過的圖片

## 此 Skill 的功能

1. **分析圖片品質**：檢查解析度、銳利度和壓縮失真
2. **增強解析度**：智能地提升圖片解析度
3. **改善銳利度**：增強邊緣和細節
4. **減少失真**：清理壓縮失真和雜訊
5. **針對使用情境優化**：根據預期用途調整（網頁、印刷、社群媒體）

## 範例

**使用者**：「改善 screenshot-2024.png 的圖片品質」

**輸出**：
```
正在分析 screenshot-2024.png...

目前規格：
- 解析度：1920x1080
- 格式：PNG
- 品質：良好，但稍微模糊

應用的增強：
✓ 提升至 2560x1440（視網膜顯示）
✓ 銳化邊緣
✓ 增強文字清晰度
✓ 優化檔案大小

儲存為：screenshot-2024-enhanced.png
原始檔案保留為：screenshot-2024-original.png
```

**靈感來自：** Lenny Rachitsky 在他的電子報中的工作流程 - 用於他文章中的螢幕截圖

## 技巧

- 始終保留原始檔案作為備份
- 最適合螢幕截圖和數位圖片
- 可以批次處理整個資料夾
- 如需要可指定輸出格式（PNG 品質較高，JPG 檔案較小）
- 對於社群媒體，請提及平台以獲得最佳尺寸

## 常見使用案例

- **部落格文章**：發布前增強螢幕截圖
- **文件**：讓 UI 螢幕截圖清晰明瞭
- **社群媒體**：為 Twitter、LinkedIn、Instagram 優化圖片
- **簡報**：為大螢幕提升圖片解析度
- **印刷品**：為實體媒體增加解析度
