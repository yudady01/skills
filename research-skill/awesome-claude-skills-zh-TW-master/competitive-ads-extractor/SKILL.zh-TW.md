# Competitive Ads Extractor

此技能從廣告庫中提取競爭對手的廣告，並分析什麼有效 —— 他們強調的問題、針對的使用案例，以及引起共鳴的文案/創意。

## 何時使用此技能

- 研究競爭對手的廣告策略
- 為您自己的廣告尋找靈感
- 了解市場定位
- 識別成功的廣告模式
- 分析有效的訊息傳遞
- 發現新的使用案例或痛點
- 使用經過驗證的概念規劃廣告活動

## 此技能的功能

1. **提取廣告**：從 Facebook Ad Library、LinkedIn 等抓取廣告
2. **擷取螢幕截圖**：儲存所有廣告的視覺副本
3. **分析訊息**：識別問題、使用案例和價值主張
4. **分類廣告**：按主題、受眾或格式分組
5. **識別模式**：尋找常見的成功方法
6. **提供見解**：解釋為什麼某些廣告可能表現良好

## 範例

**使用者**："從 Facebook Ad Library 提取 Notion 的廣告，並告訴我什麼訊息對他們有效。"

**流程**：
```
存取 Facebook Ad Library...
搜尋：Notion
找到：23 個活躍廣告

提取螢幕截圖...
[████████████████████] 100%

儲存至：competitor-ads/notion/
- ad-001-collaboration.png
- ad-002-productivity.png
- ad-003-templates.png
...

分析訊息...
```

**輸出**：
```
# Notion 廣告分析

## 概述
- 總廣告數：23 個活躍
- 主要主題：生產力（35%）、協作（30%）、
  範本（20%）、AI 功能（15%）
- 廣告格式：靜態圖片（60%）、影片（40%）
- CTA 模式：「免費試用」、「開始使用」

## 他們強調的關鍵問題

1. **分散的資訊**（8 個廣告）
   文案：「停止在 10 個不同工具之間切換」
   為什麼有效：許多團隊面臨的直接痛點

2. **會議過載**（5 個廣告）
   文案：「用非同步更新取代不必要的會議」
   為什麼有效：COVID 後遠端工作痛點

3. **遺失的文件**（4 個廣告）
   文案：「再也不要問『那個文件在哪裡？』」
   為什麼有效：通用的職場挫折感

## 成功的創意模式

### 模式 1：前後對比
- 顯示混亂的工具環境 → 乾淨的 Notion 工作區
- 用於 6 個高效能廣告
- 視覺隱喻立即清晰

### 模式 2：功能展示
- 實際產品使用的 GIF
- 在 5 秒內顯示特定功能
- 用於新功能（AI、範本）

### 模式 3：社會證明
- 「加入 2000 萬使用者」訊息
- 客戶標誌
- 用於 4 個針對企業的廣告

## 有效的文案

最佳標題：
1. 「您團隊的知識，終於集中在一個地方」
   → 注重利益，直接解決痛點

2. 「多合一工作區」
   → 清晰的定位，廣泛吸引力

3. 「真正幫助您工作的 AI」
   → 解決 AI 懷疑，實用角度

最佳內文文案模式：
- 簡短句子（少於 10 個字）
- 專注於結果而非功能
- 包含具體數字（「減少 50% 的會議」）

## 受眾定位見解

根據廣告變體：
- 創業創始人：個人生產力角度
- 團隊負責人：協作和一致性
- 企業：提及安全性和合規性
- 學生：免費計劃、範本、組織

## 對您的廣告的建議

1. **測試「工具蔓延」痛點**
   → 根據其廣告頻率，具有強烈共鳴

2. **使用產品螢幕截圖而非抽象視覺**
   → 所有頂級廣告都顯示實際 UI

3. **以問題開頭，而非解決方案**
   → 「厭倦了 X？」比「介紹 Y」表現更好

4. **保持文案在 100 個字元以下**
   → 他們最短的廣告似乎最常見

5. **測試前後視覺格式**
   → 在其創意中經過驗證的模式

## 儲存的檔案
- 所有廣告：~/competitor-ads/notion/
- 分析：~/competitor-ads/notion/analysis.md
- 最佳表現者：~/competitor-ads/notion/top-10/
```

## 您可以學到什麼

### 訊息分析
- 他們強調的問題
- 如何相對於競爭對手定位
- 引起共鳴的價值主張
- 目標受眾區隔

### 創意模式
- 有效的視覺風格
- 影片與靜態圖片表現
- 色彩方案和品牌
- 版面配置模式

### 文案公式
- 標題結構
- 行動呼籲模式
- 長度和語調
- 情感觸發

### 活動策略
- 季節性活動
- 產品發布方法
- 功能公告策略
- 重定向模式

## 最佳實踐

### 法律與道德
✓ 僅用於研究和靈感
✓ 不要直接複製廣告
✓ 尊重智慧財產權
✓ 使用見解來告知原創創意
✗ 不要抄襲文案或竊取設計

### 分析技巧
1. **尋找模式**：哪些主題重複？
2. **隨時間追蹤**：每月儲存廣告以查看演變
3. **測試假設**：為您的品牌調整成功模式
4. **按受眾區隔**：不同目標的不同訊息
5. **比較平台**：LinkedIn 與 Facebook 訊息不同

## 常見工作流程

### 廣告活動規劃
1. 提取競爭對手廣告
2. 識別成功模式
3. 注意其訊息中的空白
4. 腦力激盪獨特角度
5. 草擬測試廣告變體

### 定位研究
1. 從 5 個競爭對手獲取廣告
2. 繪製其定位
3. 尋找服務不足的角度
4. 開發差異化訊息
5. 與其方法進行測試

### 創意靈感
1. 按主題提取廣告
2. 分析視覺模式
3. 注意色彩和版面配置趨勢
4. 調整成功模式
5. 建立原創變體

## 成功提示

1. **定期監控**：每月檢查變化
2. **廣泛研究**：也查看相鄰競爭對手
3. **儲存所有內容**：建立參考庫
4. **測試見解**：執行您自己的實驗
5. **追蹤表現**：A/B 測試受啟發的概念
6. **保持原創**：用於靈感，而非複製
7. **多個平台**：比較 Facebook、LinkedIn、TikTok 等

## 輸出格式

- **螢幕截圖**：所有廣告儲存為圖片
- **分析報告**：見解的 Markdown 摘要
- **試算表**：包含廣告文案、CTA、主題的 CSV
- **簡報**：頂級表現者的視覺簡報
- **模式庫**：按方法分類
