# Documentation Templates

Index of all available templates. Load specific template when needed.

## Screen Templates

For screen documentation under `docs/screens/{screen-name}/`:

- **[screen-readme.md](./screen-readme.md)** - Screen overview template (max 300 lines)
- **[screen-technical.md](./screen-technical.md)** - Technical implementation template (max 500 lines)
- **[screen-features.md](./screen-features.md)** - Detailed features template (max 400 lines)
- **[screen-flows.md](./screen-flows.md)** - User flows template (max 300 lines)

## Architecture Templates

For architecture docs under `docs/architecture/`:

- **[architecture.md](./architecture.md)** - Architecture documentation template (max 500 lines)

## Guide Templates

For guides under `docs/guides/`:

- **[guide.md](./guide.md)** - User/developer guide template (max 500 lines)

## Usage

1. Determine document type
2. Select appropriate template from list above
3. Load specific template file
4. Replace placeholder variables
5. Fill in content

## Placeholder Variables

All templates use these placeholders:

- `{Screen Name}` - Capitalized (e.g., "Home", "Config")
- `{screen-name}` - Lowercase with hyphens (e.g., "home", "config")
- `{Architecture Topic}` - Topic name (e.g., "3-Layer Architecture")
- `{Guide Name}` - Guide name (e.g., "User Guide")
