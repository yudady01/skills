## Guide Types

**User Guide**: For end users
- Focus on "how to use"
- Simple language
- Many examples

**Developer Guide**: For contributors
- Focus on "how to build/modify"
- Technical details
- Code standards

**Configuration Guide**: For setup
- Focus on options and values
- Tables for parameters
- Examples for each option
