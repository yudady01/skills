# 代码质量命令

这个命令执行全面的代码质量检查，确保代码符合最高的质量标准。

## 质量检查项目

### 1. 类型安全
- TypeScript 严格模式检查
- 类型定义完整性
- 接口契约验证

### 2. 代码规范
- 代码格式检查
- 命名规范验证
- 编码标准合规

### 3. 测试质量
- 单元测试覆盖率
- 集成测试完整性
- 边界条件测试

### 4. 性能分析
- 算法复杂度分析
- 内存使用检查
- 性能回归检测

### 5. 安全检查
- 安全漏洞扫描
- 依赖安全审计
- 输入验证检查

## 执行命令

```bash
# 运行所有质量检查
/code-quality

# 运行特定类型检查
/code-quality --type-check

# 运行测试覆盖率检查
/code-quality --coverage

# 运行安全检查
/code-quality --security
```

通过这个命令，您可以确保代码符合所有质量标准和最佳实践要求。
