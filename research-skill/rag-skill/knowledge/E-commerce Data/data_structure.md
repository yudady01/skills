# 领域目录索引：E-commerce Data

本目录用于存放电商业务相关的结构化数据文件（当前以 CSV 为主），常用于用户分析、销售分析和库存管理等场景。

- employees.xlsx
  - 用途：员工基本信息和相关属性数据，可用于组织结构分析、人力相关统计等。
- customers.xlsx
  - 用途：客户/用户数据，包括基本属性和可能的分群字段，是用户画像与运营分析的基础数据。
- inventory.xlsx
  - 用途：商品库存相关数据，用于库存周转分析、缺货预警等。
- sales_orders.xlsx
  - 用途：销售订单明细数据，可用于销售额统计、转化分析、复购分析等。

如有新增其他电商相关数据文件（如商品信息、流量日志等），可放在本目录并在此文件中补充说明用途。
